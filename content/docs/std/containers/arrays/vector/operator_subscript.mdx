Returns a reference to the element at specified index `pos`. No bounds checking is performed.

:::note
The types `reference` and `const_reference` are explained in
[std::vector class reference](../#type-names)
:::

### Parameters

- `pos` - position of the element to return

### Return value

Reference to the requested element.

### Exceptions

None.

### Complexity

Constant.

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <vector>
#include <iostream>
 
int main()
{
	std::vector<int> numbers {2, 4, 6, 8};

	std::cout << "Second element: " << numbers[1] << '\n';

	numbers[0] = 5;

	std::cout << "All numbers:";
	for (auto i : numbers) {
		std::cout << ' ' << i;
	}
	std::cout << '\n';
}
```
