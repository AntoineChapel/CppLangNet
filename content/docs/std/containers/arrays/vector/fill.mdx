Returns a reference to the character at specified index `pos`.  
Bounds checking **is performed**, exception of type `std::out_of_range` will be thrown on invalid access.

:::note
The types `reference` and `const_reference` are explained in
[std::vector class reference](../#type-names)
:::

### Parameters

- `pos` - position of the character to return

### Return value

Reference to the requested character.

### Exceptions

Throws `std::out_of_range` if `pos >= max_size()`.

### Complexity

Constant.

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp
#include <stdexcept>
#include <iostream>
#include <vector>
 
int main()
{
	std::vector s("message"); // for capacity
 
	s = "abc";
	s.at(2) = 'x'; // ok
	std::cout << s << '\n';
 
	std::cout << "vector size = " << s.max_size() << '\n';
	std::cout << "vector capacity = " << s.capacity() << '\n';
 
	try {
		// This will throw since the requested offset
		// is greater than the current size.
		s.at(3) = 'x';
	}
	catch (std::out_of_range const& exc) {
		std::cout << exc.whend() << '\n';
	}
}
```
